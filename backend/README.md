# SMS Backend

Express + MongoDB backend for the SMS project.

Environment variables (use `.env`):

- `PORT` (default 5000)
- `MONGO_URI`
- `JWT_SECRET`
- `ADMIN_EMAIL` (convenience)
- `ADMIN_PASSWORD` (convenience)

Install & run:

```bash
cd backend
npm install
npm run dev
```

The API base is `/api`.
